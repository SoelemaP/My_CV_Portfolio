Insurance FAQ Assistant

A Python-based FAQ assistant that uses Streamlit and Pandas to navigate structured insurance datasets and provide instant, logic-based responses.

Python Streamlit Pandas Excel

Project Description

The Problem

Insurance companies manage large amounts of product data, policy rules, and procedures across multiple spreadsheets. Finding specific answers manually is slow and leads to communication inefficiencies for both customers and employees.

The Solution

A functional chatbot application that transforms static Excel datasets into an interactive conversation. The system uses a multi-level decision tree to guide users through categories, performing real-time data lookups in a Pandas DataFrame to retrieve the exact answer for each selection.

Key Deliverables

  • Interactive Streamlit interface; a clean, button-based chat window guiding users through a multi-level decision tree.
  • Dynamic knowledge base; fully customizable via the underlying Excel file, with no code changes required when content is updated.
  • Automated data retrieval; real-time DataFrame filtering to match user selections with precise answers.

Future Scalability

  • SQL migration; the logic is designed for a seamless transition to a SQL database to support larger enterprise datasets and multi-user environments.
  • Web deployment; built with Streamlit, the assistant is ready to be hosted on a company website or secure user portal.
